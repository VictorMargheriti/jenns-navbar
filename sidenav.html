Do you like my effects on the buttons on my sidebar?

If you do, you can add this to your anchor tags in the CSS file:
ul li a {
transition: all .4s;
}
ul li a:hover {
transform: translateX(10px);
}
adding :hover to the element will apply those changes when you hover your mouse and adding the .4s transition makes it look smooth and animated
